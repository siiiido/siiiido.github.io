### fetch() : 백엔드로부터 데이터를 받아오려면 api를 호출하고 데이터를 응답받아야 하는데 그 때 fetch() 함수 사용, 비동기 HTTP 요청 또는 axios 라이브러리 사용

fetch().then().then() 형식을 기억하고   
fetch() 함수 기본 method는 get  
fetch와 axios는 결국 라이브러리다 → 중요한건 http 통신의 요청과 응답에 대한 이해와 promise 개념이다.~~
