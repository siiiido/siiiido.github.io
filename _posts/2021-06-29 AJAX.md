JSON(Javascript Object Notation) : javascript 객체를 만드는 표현식
* key:value 형식의 데이터 저장 방식


AJAX(Asynchronous Javascript and XML) : 비동기 형식의 통신 즉 전체 페이지를 로드하는 동기식 통신이 아닌 일부분의 데이터만 변한 것을 로드해줌(좋아요를 눌렀을 때 페이지는 새로고침 되지 않았는데 좋아요 갯수만 올라간 것)
